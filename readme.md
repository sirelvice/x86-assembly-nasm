# x86 Assembly Language – Learning & Experiments

This repository documents my personal exploration of x86 Assembly
Language using NASM.

The goal is to understand low-level programming concepts such as:
- CPU registers and flags
- Instruction execution flow
- Arithmetic and logic at the hardware level
- Control flow, loops, and comparisons
- Function calls and calling conventions
- Basic input/output using system calls

This is a hands-on learning space focused on clarity, experimentation,
and building strong mental models of how programs execute at the CPU level.

## Tools
- NASM (x86, 32-bit)
- Linux / WSL
- VS Code
- GDB (optional, for debugging)

## Structure
- `docs/` – Concept explanations and notes
- `programs/` – Small focused assembly programs
- `tooling/` – Build, run, and debugging notes
